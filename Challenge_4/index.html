Hypertext Markup Language

HTML is the standard document structuring language will use to make web pages. Web applications will run on the web thus HTML will help us create the content and layout of our web application.

Cascading Stylesheet

Cascading Style Sheets (CSS) is a style sheet language used for describing the look and feel of a document written in a markup language like HTML. CSS is design focused making our web pages more interesting and fun to view and engage with.

JavaScript

JavaScript is a high-level, interpreted, object-oriented programming language. It is a scripting language that we'll use to add functionality and interaction with the page's elements or content. It is often called Programming Language of the web.
